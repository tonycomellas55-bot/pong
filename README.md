# Pong (Vibe)

Run it:

```bash
cd pong
bun run build
bun run serve
```

Then open:

- `http://127.0.0.1:5173/`

Dev loop:

- Edit `pong/src/main.ts`
- Watch build: `cd pong && bun run dev`
- Serve: `cd pong && bun run serve`

Controls:

- Move: `W/S` or `Up/Down`
- Pause: `Space`
- Serve: `Enter` (or tap/click)
- Reset: `R`
